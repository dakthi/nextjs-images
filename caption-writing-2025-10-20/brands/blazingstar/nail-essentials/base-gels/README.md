# BlazingStar Base Gels

## Overview
Foundation products for gel polish and builder gel applications. Each base offers specific benefits for different nail types and application styles.

## Base Gel Options

### 1. Original Gel Base
**Characteristics:**
- **Texture:** Thin, easy to apply
- **Benefit:** Contains nail care components
- **Best for:** Natural nails, everyday wear
- **Sizes:** 15ml, 240ml

**Application:**
- Layer before color for protection
- Helps color adhere better
- Nourishes and protects natural nail

### 2. Revive Base
**Characteristics:**
- **Texture:** Thicker, more substantial
- **Benefit:** Stronger foundation for durability
- **Best for:** Gel polish application, builder gel base
- **Sizes:** 15ml, 240ml

**Application:**
- Dual-purpose base for gel and builder
- Provides firm adhesion
- Perfect for extended wear applications

### 3. TPO/HEMA-Free Base Gel
**Characteristics:**
- **Formula:** Allergen-reduced formula
- **Benefit:** Safe for sensitive clients
- **Best for:** Clients with allergies, sensitive skin
- **Sizes:** 15ml

**Application:**
- Essential for allergy-prone clients
- Maintains durability without harsh chemicals
- Supports nail health

## Choosing the Right Base

**For Natural Nails:**
→ Original Gel Base (nourishing, protective)

**For Gel Polish:**
→ Revive Base or Original (provides adhesion)

**For Builder Gel Application:**
→ Revive Base (stronger foundation)

**For Sensitive Clients:**
→ TPO/HEMA-Free Base (allergen-safe)

## Key Benefits

✓ **Protection:** Shields natural nail from staining
✓ **Adhesion:** Improves color adherence and durability
✓ **Health:** Nourishing formulas support nail health
✓ **Safety:** Hypoallergenic options for sensitive clients
✓ **Versatility:** Works with both gel polish and builder systems
