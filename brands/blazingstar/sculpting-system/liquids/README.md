## Overview / Tổng Quan
Professional-grade monomer liquids designed for optimal acrylic powder application. Each liquid offers specific working characteristics to match your technique, environment, and speed requirements.

Dung dịch monomer chuyên nghiệp được thiết kế cho ứng dụng bột acrylic tối ưu. Mỗi dung dịch cung cấp các đặc tính làm việc cụ thể để phù hợp với kỹ thuật, môi trường và yêu cầu tốc độ của bạn.

## Temperature & Speed Optimization / Tối Ưu Hóa Nhiệt Độ & Tốc Độ

**Working with Powder Speed Selection / Làm Việc Với Lựa Chọn Tốc Độ Bột:**

All liquids work with BlazingStar's 3-speed powder system:
- **Hot Weather (Summer):** Use Slow Set powder / **Thời Tiết Nóng (Mùa Hè):** Dùng bột Khô Chậm
- **Moderate Weather (Spring/Fall):** Use Medium Set powder / **Thời Tiết Vừa (Xuân/Thu):** Dùng bột Khô Vừa
- **Cold Weather (Winter):** Use Fast Set powder / **Thời Tiết Lạnh (Mùa Đông):** Dùng bột Khô Nhanh

This system allows you to maintain consistent working time regardless of temperature / Hệ thống này cho phép bạn duy trì thời gian làm việc nhất quán bất kể nhiệt độ

## Application Tips / Mẹo Ứng Dụng

1. **Proper ratio** / **Tỷ lệ đúng:** Mix correct powder-to-liquid ratio for optimal consistency / Trộn tỷ lệ bột-dung dịch đúng để độ đặc tối ưu
2. **Room temperature** / **Nhiệt độ phòng:** Store at room temperature for best performance / Bảo quản ở nhiệt độ phòng để hiệu suất tốt nhất
3. **Work quickly** / **Làm nhanh:** Once bead is formed, work efficiently / Khi hạt được tạo thành, làm việc hiệu quả
4. **Clean tools** / **Làm sạch dụng cụ:** Clean brushes immediately after use / Làm sạch cọ ngay sau khi sử dụng
5. **Ventilation** / **Thông gió:** Ensure proper ventilation in salon / Đảm bảo thông gió thích hợp trong salon
